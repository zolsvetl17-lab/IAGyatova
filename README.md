# Movie Library

**Автор:** Милена Гятова

## Описание

Movie Library — это десктопное GUI‑приложение на Python для организации личной коллекции фильмов. Программа позволяет:

* добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга;
* просматривать все добавленные фильмы в виде таблицы;
* фильтровать записи по жанру и году выпуска;
* сохранять данные в формате JSON между сессиями работы;
* проверять корректность вводимых данных (год, рейтинг).

## Технические детали

* **Язык программирования:** Python 3.x
* **GUI‑библиотека:** Tkinter (встроенная в Python)
* **Формат хранения данных:** JSON (`movies.json`)
* **Система контроля версий:** Git

## Требования

Для запуска приложения необходим установленный **Python 3.6+** и стандартные библиотеки:
* `tkinter` (обычно входит в поставку Python);
* `json` (встроенная библиотека).

Никаких дополнительных установок не требуется.

## Инструкция по использованию
### Добавление фильма
1. Перейдите на вкладку **«Добавить фильм»**.

2. Заполните поля:
* **Название** — введите название фильма.
* **Жанр** — укажите жанр (например, «Драма», «Фантастика»).
* **Год выпуска** — введите год (число от 1800 до 2100).
* **Рейтинг** — поставьте оценку от 0 до 10.

3. Нажмите кнопку **«Добавить фильм»**. Если данные корректны, фильм появится в таблице. При ошибке появится сообщение с указанием проблемы.

### Фильтрация фильмов
1. Перейдите на вкладку **«Фильтр»**.

2. Введите:
* **Жанр** (например, «Комедия») — для фильтрации по жанру.

* **Год** (например, 2020) — для фильтрации по году выпуска.

3. Нажмите **«Применить фильтр»** — таблица обновится, отображая только подходящие записи.

## Просмотр всех фильмов
Чтобы сбросить фильтры и увидеть всю коллекцию, очистите поля фильтрации и нажмите **«Применить фильтр»**.

## Проверка корректности данных
### Программа автоматически проверяет:

* **Год выпуска**: должен быть числом в диапазоне 1800–2100.
* **Рейтинг**: должен быть числом от 0 до 10 (допускаются десятичные дроби, например, 8.5).
* **Название и жанр**: не могут быть пустыми.

При нарушении этих правил выводится сообщение об ошибке.
